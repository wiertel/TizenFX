    Methods

    Add(View)

    Adds a child view to this Container.

    Declaration
    public abstract void Add(View view)
    Parameters
    Type Name Description
    View view

    The child view to add.
    Remarks

    If the child already has a parent, it will be removed from the old parent and reparented to this view. This may change child's position, color, scale, etc. as it now inherits them from this view.

    API Version
    4
    Precondition

    This Container (the parent) has been initialized. The child view has been initialized. The child view is not the same as the parent view.

    Postcondition

    The child will be referenced by its parent. This means that the child will be kept alive, even if the handle passed into this method is reset or destroyed.
